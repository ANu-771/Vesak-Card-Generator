# 🪔 Vesak Card Generator

A creative and interactive **Vesak Card Generator** built using **HTML, CSS, and JavaScript** that allows users to create personalized digital Vesak greeting cards with an attractive and responsive user interface.

This project focuses on celebrating Vesak traditions through modern web technologies by providing an easy way to generate and customize greeting cards.

---

## 📌 Overview

The Vesak Card Generator is a frontend web application developed to allow users to design and generate digital Vesak greeting cards. The application provides an interactive experience through custom designs, responsive layouts, and dynamic card generation features.

This project demonstrates frontend development concepts, user interaction handling, and creative UI design.
